One-Line Website Include

Add this script once in the shared footer or common template used by every page. The loader injects the widget CSS and JavaScript automatically.

<script src="http://localhost:8000/embed.js" data-api-url="http://localhost:8000" data-kwekwe-widget defer></script>

Integration Notes

The widget is designed for PHP-based pages and other static templates. Add the script include to the global footer, layout, or reusable include file to make it appear on every page automatically.
