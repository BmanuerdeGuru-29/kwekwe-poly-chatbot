This is a demonstration page showing how the chat widget integrates with the Kwekwe Polytechnic website. The chat widget appears as a floating button in the bottom-right corner of this page.

To test the chat widget:

  1. Look for the floating chat button in the bottom-right corner
  2. Click the button to open the chat window
  3. Try asking questions about Kwekwe Polytechnic programs, fees, or services

Integration Information

The chat widget can be easily integrated into any website with just two lines of code:

CSS (add to head):

<link rel="stylesheet" href="kwekwe-chat-widget.css">

JavaScript (add before closing body tag):

<script src="kwekwe-chat-widget.js"></script>
Note: The widget automatically loads Font Awesome icons and initializes itself when the page loads. No additional configuration is required.
